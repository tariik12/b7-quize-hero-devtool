map, filter,forEach and find are all built in array methods in Java-Script. But there have some difference:
.map()
executes the same code on every element in an array and returns a new array with the updated elements.
Example:
const number = [1, 2, 3, 4, 5]; const doubled = number.map(num => num * 2); console.log(doubled) //output: [2, 4, 6, 8, 10]

.filter()
checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria.
Example:
const number = [1, 2, 3, 4, 5]; const even = nums.filter(num => num % 2 === 0); // console.log(even) //output: [2, 4]

.forEach()
is used to execute the same code on every element in an array but does not change the array and it returns undefined.
Example:
const number = [1, 2, 3, 4, 5]; number.forEach(num => console.log(num)); //output: 1, 2, 3, 4, 5

.find()
is used to find the first element in an array that meets a condition and return it.
Example:
const number = [1, 2, 3, 4, 5]; const firstEven = number.find(num => num % 2 === 0); // 2 //output: 2

Local Storage, Session Storage , and Cookies are all ways for websites to store information on the user's browser, but they differ in their storage capacity, lifespan, and accessibility.

Local Storage is a type of storage that allows websites to store data on the user's device indefinitely, even after the browser window is closed. The amount of data that can be stored in Local Storage is generally much larger than that of Session Storage or Cookies. This data can be accessed by any page on the same website.

Session Storage is a type of storage that allows websites to store data for the duration of a session, which ends when the user closes the browser window. The amount of data that can be stored in Session Storage is generally smaller than that of Local Storage, but larger than that of Cookies. This data can also be accessed by any page on the same website.

Cookies are small text files that websites can store on the user's device. The amount of data that can be stored in Cookies is generally smaller than that of both Local Storage and Session Storage. Cookies can have an expiration date set by the website, or they can be set to expire when the user closes the browser window. Cookies can be accessed by any page on the same website, as well as by third-party websites that the user visits
